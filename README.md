# Website

A Jekyll implementation of the [Creative Theme](http://startbootstrap.com/template-overviews/creative/) template. 

- Start by adding your info in `_config.yml`
- In `_layouts/front.html` reorder or remove section as needed

# Running locally
Ensure Ruby is installed on your machine (check with `ruby -v`). To use Gemfile, the Bundler gem is also necessary (`gem install bundler`).

To start the jekyll server: `bundle exec jekyll serve`
